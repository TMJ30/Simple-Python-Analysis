# PyBank PyPoll Analysis

**PyBank:** Created a Python script to analyze financial records of a sample company, PyBank, over time using the financial dataset provided in the [budget data](PyBank/Resources/budget_data.csv) CSV file. The script calculates key metrics to provide insights into the comapny's financial performance.

The analysis includes:
* Total number of months in the datset
* The total net smount of **Profit/Losses** over the entire period
* The month-to-month changes in **Profit/Losses** and average of those changes
* The greatest increase in profits (with date and amount)
* The greatest decrease in profits (with date and amount)


**PyPoll:** Created a Python script to analyze the voting records for a local election using the election dataset provided in the [election data](PyBank/Resources/election_data.csv) CSV file. The script calculates key metrics to provide insights into the election outcome.

The analysis includes:
* The total number of votes cast
* A complete list of candidates who received votes
* The percentage of votes each candidate won
* The total number of votes each candidate won
* The winner of the election based on popular vote
